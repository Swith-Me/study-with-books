한 작업을 정리해보면

1. 오래된 주석을 간단하게 고치고 개선했다. 
2. enum을 모두 독자적인 소스 파일로 옮겼다   
3. 정적 변수와 정적 메서드를 새 클래스로 옮겼다
4. 일부 추상 메서드를 DayDate 클래스로 끌어올렸다
5. Month.make를 Month.fromInt로 변경했다

7. 팔방미인으로 사용되던 숫자 1을 없앴다
