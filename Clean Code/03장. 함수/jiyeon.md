
## 3장 함수

### 가장 기본적인 단위 : 함수
------------
### 작게 만들어라!
* 함수를 만드는 첫번째 규칙 : 작게 만든 것
* 함수를 만드는 두번째 규칙 : 더 작게 만드는 것
* 함수는 가로 150자를 넘겨선 안되고, 100줄을 넘겨서도 안된다. (20줄도 긺)

#### 블록과 들여쓰기
* if문, else문, while문에 들어가는 블록들은 한 줄이어야 한다
* 중첩 구조가 생길 만큼 함수가 커져서는 안되기 대문에 들여쓰기는 1단이나 2단을 넘어서면 안된다
